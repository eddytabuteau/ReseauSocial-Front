Bonjour à tous !

Je m'appelle Eddy et je suis le concepteur du site Reseau Social.

Le site est développé côté Front avec Angular et pour le back avec Node JS.

Pour stocker les données, j'ai utilisé la base de donnée MongoDB.

Les principales fonctions de ce "réseau social" sont les suivantes :

- Rajouter des utilisateurs à votre liste d'amis

- Proposer des amis à vos amis

- Lancer une discussion instannée avec un ami

- Avoir un espace avec votre profil

- Publier des messages sur votre profil ainsi que ceux de vos amis

- Lancer un chat sur le sujet que vous voulez avec les amis souhaités
